What is SOMAH2 ?

A culturally sensitive communication app for breastfeeding and infant feeding counselling

The SOMAH2 project aims to increase breastfeeding practice of new mothers by providing a medical application that will enhance the communication with midwives.

The application was developped during the spring of the European Project Semester (EPS) 2016 in Oslo and Akershus University College. It was conceptualized using a collaborative process with public health researchers.

The code is entirely open source and available on Github.

If you need more information, a complete report of the project is available here.
